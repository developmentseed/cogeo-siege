## Install

You can install `cogeo-siege` using pip

```bash
$ pip install -U pip
$ pip install -U cogeo-siege
```

or install from source:

```bash
$ git clone https://github.com/developmentseed/cogeo-siege.git
$ cd cogeo-siege
$ pip install -U pip
$ pip install -e .
```
